# 🌭 Hotdog Van

Street Food Tycoon game built with React + Vite.

## Features
- 60-second daily gameplay
- Grill sausages (they burn if left too long!)
- Serve customers with random orders (Plain, Ketchup, Mustard)
- Upgrade shop between days
- Combo system for bonus earnings

## Run
```bash
pnpm install
pnpm dev
```
